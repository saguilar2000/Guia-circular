# Modos de Propagación en Guías de Onda Cilíndricas
Este repositorio contiene un script escrito en Python para la simulación y visualización de los campos de los Modos Transversales Eléctricos (TE) y Transversales Magnéticos (TM) que se propagan dentro de una Guía de Onda Cilíndrica (sección circular) perfectamente conductora y vacía.

El cálculo de los campos se basa en la solución de las ecuaciones de Maxwell en coordenadas cilíndricas, utilizando las funciones de Bessel.
